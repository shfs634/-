核心技术与功能

Core Technology

三类独立存储

Triple Compartments

支持可回收物、厨余垃圾、其他垃圾三类独立密封存放，支持智能感应开盖，防止异味扩散。

Supports independent sealed storage for Recyclables, Kitchen, and Other waste with smart sensor lids.

AI 视觉识别

AI Vision System

分辨垃圾颜色来确定投放种类，自动执行分类存储动作。具备语音引导与LCD屏幕说明。

Identifies waste by color to determine category and executes sorting. Includes voice guidance and LCD display.

安全交互机制

Safety Interaction

确保用户在场且与机器保持10cm安全距离，全自动运行，无需人工干预。

Ensures user presence at 10cm safety distance. Fully automated operation without intervention.

L610 物联网

L610 IoT System

通过手机端实时监控。空间不足时通过L610实时上报物业中心告警并提示更换。

Real-time mobile monitoring. L610 reports alerts to management cloud when storage is full.

系统运行模式

System Operation Modes

Controlled by Android host computer with LCD parameter settings. Includes:

  • 1. 操作模式（常规识别与投放）
  • 2. 维护模式（系统检查与设置）
  • 1. Operation Mode (Standard sorting)
  • 2. Maintenance Mode (System diagnosis)
